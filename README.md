# Customer Segmentation & Spending Analysis Dashboard

## Overview
This project analyzes customer demographics, income distribution, marital status, education levels, and wine spending behavior using Microsoft Excel.

## KPIs
- Total Customers
- Total Income
- Average Income
- Total Wine Spending
- Average Recency

## Dashboard Visualizations
- Income Distribution by Education Level
- Income Share by Marital Status
- Wine Spending by Education Level
- Customer Demographics Analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- KPI Cards
- Data Cleaning
## Key Insights
- Married customers contribute a significant share of income.
- Education level impacts spending behavior.
- Wine spending differs across customer groups.
